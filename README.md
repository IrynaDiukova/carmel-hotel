# Carmel Hotel — стандарты работы сотрудников

Интерактивный справочник для новых сотрудников отеля **Кармель**. Пользователь
листает карточки: правила, коды номеров, примеры заданий в формате WhatsApp,
проверка и подготовка номеров, конец смены. Навигация — только **Назад** и
**Далее**.

Работает как:
- **PWA** — мобильное веб-приложение (компьютер, планшет, телефон; офлайн);
- **Telegram Web App** — родные кнопки Back / Main (SDK грузится только в Telegram).

## Что внутри

| Файл | Назначение |
|------|------------|
| `index.html` | Всё приложение: разметка, стили, логика, контент всех экранов |
| `carmel-logo.svg` | Логотип-эмблема Carmel Hotel (вектор) |
| `icon.svg` | Иконка приложения (PWA) |
| `manifest.webmanifest` | Манифест PWA |
| `sw.js` | Service Worker — офлайн-кэш (HTML — network-first, статика — cache-first) |

Сборка не нужна — это статика на чистом HTML/CSS/JS.

## Оформление

- Светло-голубой фон, белые карточки, чёрный текст, бирюзовый акцент Carmel.
- Примеры заданий — окошки в стиле **WhatsApp**: входящее «Описание заданий» (белый
  пузырь) с реакцией 👍, ответ «от меня» (зелёный пузырь), курсивное пояснение.
- Примеры **SMS** менеджеру, коды номеров столбиком, цветные рамки (важно / запрет /
  инфо), чек-листы.
- Прогресс-бар, счётчик «X / N», сохранение позиции (`localStorage`), свайпы и стрелки.

## Коды номеров

Коды на экране «Коды для открытия номеров» — это **иллюстративные примеры**, не реальные,
поэтому публичная ссылка безопасна и приватная версия не требуется.

## Запуск локально

```powershell
python -m http.server 8080
```
Открой <http://localhost:8080>. На телефоне в той же сети — `http://<IP-ПК>:8080`.

## Публикация на GitHub Pages (ссылка вида carmel-hotel)

1. Создай на github.com репозиторий с именем **`carmel-hotel`** (Public).
2. Залей файлы из этой папки (drag-and-drop в «uploading an existing file») **или**
   запушь существующий git-репозиторий:
   ```powershell
   cd "E:\Новая папка\Git\bin\carmel-hotel"
   git remote add origin https://github.com/ТВОЙ_ЛОГИН/carmel-hotel.git
   git push -u origin main
   ```
3. **Settings → Pages** → ветка `main` / `/(root)` → Save.
4. Ссылка: `https://ТВОЙ_ЛОГИН.github.io/carmel-hotel/`

## Где менять контент

Весь контент — массив `SCREENS` в `index.html`. Один экран:
```js
{ sec:"Задания", icon:"🧽", title:"Заголовок", blocks:[ ...блоки... ] }
```

Блоки: `{p}` абзац (`*жирный*`), `{h}` подзаголовок, `{list, ordered}` список,
`{wa:[{in,react},{out}], head}` окно WhatsApp, `{sms}` пример SMS, `{expl}` курсивное
пояснение, `{pins:[["1620","4114"],…]}` коды, `{box:"info|warn|danger", title, blocks}`
цветная рамка, `{check:[…]}` чек-лист. Первый экран — `{center:true}` с `{logo}`,
`{hero}`, `{sub}`. Счётчик «/ N» обновляется автоматически.

## Заменить логотип на точный оригинал

Логотип сейчас — векторная реконструкция (`carmel-logo.svg`). Чтобы поставить
оригинальный файл: положи его рядом как `carmel-logo.png` (или другой) и поменяй
`src="carmel-logo.svg"` в `index.html` (блок `{logo}` в `renderBlocks` и финальный
экран) на имя файла.
