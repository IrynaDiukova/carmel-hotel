# Онбординг персонала — отель Моше

Интерактивный справочник-тренажёр для новых сотрудников (горничных / раннеров).
Пользователь листает карточки: правила, полезные фразы, коды комнат и примеры
заданий с форматом ответа менеджеру. Навигация — **Назад**, **Далее** и
подтверждение прочтения кнопкой **ok**.

Работает как:
- **PWA** — мобильное веб-приложение (можно «установить» на телефон, работает офлайн);
- **Telegram Web App** — подхватывает тему Telegram, родные кнопки Back / Main и
  тактильный отклик (SDK Telegram грузится только при запуске из Telegram).

## Что внутри

| Файл | Назначение |
|------|------------|
| `index.html` | Всё приложение: разметка, стили, логика, контент всех экранов |
| `manifest.webmanifest` | Манифест PWA (имя, иконка, цвета, режим standalone) |
| `sw.js` | Service Worker — офлайн-кэш (HTML — network-first, статика — cache-first) |
| `icon.svg` | Иконка приложения |

Сборка не нужна — это статика на чистом HTML/CSS/JS.

## Возможности

- Вступление и правила → фразы на иврите → коды комнат → примеры заданий → финал.
- Примеры заданий показаны как чат: **«Задание от Моше»** (серый пузырь) → **«Твой
  ответ»** (бирюзовый) → пояснение «что это значит».
- Кнопки **Назад / ok / Далее**, прогресс-бар и счётчик «X / N».
- Кнопка **ok** отмечает карточку как прочитанную (сохраняется в браузере).
- Навигация свайпами (← →) и стрелками клавиатуры; Enter / пробел = «ok».
- Прогресс и отметки сохраняются между запусками (`localStorage`).
- Светлая/тёмная тема автоматически; в Telegram — по теме мессенджера.
- Кнопка **В начало** на финальном экране.

## ⚠️ Важно про коды комнат и публичную ссылку

Экран «Коды от комнат» содержит **реальные пин-коды от дверей**. Если выложить
приложение по публичной ссылке, эти коды (и суммы залогов, процедуры) сможет
увидеть любой, у кого есть ссылка, — а поисковики могут её проиндексировать.

Рекомендации:
- **Лучший вариант** — раздавать приложение только через **Telegram-бота** (Web App),
  а не публичной ссылкой.
- Либо вынести коды из публичной версии / закрыть приложение паролем.
- Если коды периодически меняются — риск ниже, но ссылку всё равно держите непубличной.

## Как запустить локально

PWA и Service Worker требуют запуска по `http://`, а не `file://`.
Из папки `hotel-onboarding`:

```powershell
python -m http.server 8080
```

Откройте <http://localhost:8080>. На телефоне в той же сети — `http://<IP-ПК>:8080`.

> Открыть `index.html` двойным щелчком тоже можно — приложение работает, но без
> офлайн-кэша (Service Worker под `file://` не регистрируется).

## Публикация (получить ссылку)

Залейте все файлы как есть на статический хостинг с HTTPS — GitHub Pages, Netlify,
Vercel, Cloudflare Pages. HTTPS обязателен для установки PWA и для Telegram Web App.
См. предупреждение о кодах комнат выше.

## Подключение к Telegram

1. Опубликуйте приложение по HTTPS-адресу.
2. У [@BotFather](https://t.me/BotFather): `/newapp` (или *Bot Settings → Web App*)
   и укажите URL.
3. Откройте Web App через кнопку бота / меню — интерфейс подхватит тему Telegram и
   родные кнопки.

## Где менять контент

Весь контент — массив `SCREENS` в `<script>` внутри `index.html`. Один экран:

```js
{ sec:"Пример задания", icon:"🔑", title:"Заголовок", blocks:[ ...блоки... ] }
```

Доступные блоки внутри `blocks`:

| Блок | Что рисует |
|------|------------|
| `{ h:"…" }` | подзаголовок |
| `{ p:"…" }` | абзац (поддерживает `\n` и `*жирный*`) |
| `{ list:[…], ordered:true }` | список (нумерованный, без `ordered` — маркированный) |
| `{ task:"…" }` | входящее задание от Моше (серый пузырь слева) |
| `{ reply:"…" }` | твой ответ (бирюзовый пузырь справа) |
| `{ msg:"…" }` | что ты пишешь сам (бирюзовый пузырь справа) |
| `{ note:"…", alert:true }` | пояснение (с `alert:true` — красная плашка «звони Моше») |
| `{ callout:"…" }` | выделенная важная строка |
| `{ phrases:[["ивр","рус"], …] }` | список фраз на иврите |
| `{ pins:[["1620","4114"], …] }` | таблица «комната → код» |

`sec` — короткая «рубрика» над заголовком (например «Правила», «Пример задания»).
Чтобы добавить экран — добавьте объект в массив; счётчик «/ N» обновится сам.
